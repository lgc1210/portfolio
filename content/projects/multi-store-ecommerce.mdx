# Multi-store e-commerce platform

This multi-store e-commerce platform provides a complete solution for creating a
marketplace where multiple vendors can set up and manage their own stores. The
platform features a modern web interface built with Next.js and a robust backend
powered by Spring Boot, offering seamless shopping experiences for customers and
comprehensive store management tools for vendors.

## Features

- **Multi-Vendor Support**: Allow multiple vendors to create and manage their
  own stores independently
- **Product Management**: Comprehensive product catalog with categories,
  variants, and inventory tracking
- **Shopping Cart & Checkout**: Seamless shopping experience with multi-vendor
  cart and secure checkout
- **User Management**: Role-based access control for customers, vendors, and
  administrators
- **Payment Processing**: Secure payment handling with support for multiple
  payment
- **Analytics Dashboard**: Comprehensive analytics for vendors and
  administrators.
- **Responsive Design**: Mobile-friendly interface that works seamlessly across
  all devices
- **Advanced Search**: Powerful search and filtering capabilities across all
  stores

## Technologies

- **Frontend**: NextJS, Tailwind CSS, TypeScript
- **Backend**: Spring Boot
- **Database**: MySQL
- **Authentication**: JWT(JSON Web Tokens)

## Prerequisites

Before you begin, ensure you have the following installed on your machine:

- Node.js (version 18 or higher)
- npm or yarn
- Java JDK (version 11 or higher)
- Maven (version 3.6 or higher)
- MySQL Server (version 8.0 or higher)

## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/lgc1210/multi-store-ecommerce.git
cd multi-store-ecommerce
```

### 2. Backend Setup (Spring Boot)

Navigate to the backend directory:

```bash
cd backend
```

Configure your database connection in application.properties:

```bash
# Database Configuration
spring.datasource.url=jdbc:mysql://localhost:3306/multistore_ecommerce
spring.datasource.username=your_username
spring.datasource.password=your_password
spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver

# JPA Configuration
spring.jpa.hibernate.ddl-auto=update
spring.jpa.show-sql=true
spring.jpa.properties.hibernate.dialect=org.hibernate.dialect.MySQL8Dialect

# JWT Configuration
jwt.secret=your-jwt-secret-key
jwt.expiration=86400000

# File Upload Configuration
spring.servlet.multipart.max-file-size=10MB
spring.servlet.multipart.max-request-size=10MB
```

Build and run the Spring Boot application:

```bash
./mvnw clean install
./mvnw spring-boot:run
```

The API will be available at http://localhost:8080

### 3. Frontend Setup (Next.js)

Open a new terminal and navigate to the frontend directory:

```bash
cd frontend
npm install
```

Create a .env.local file with your configuration:

```bash
NEXT_PUBLIC_API_BASE_URL=http://localhost:8080/api
NEXT_PUBLIC_APP_NAME=Multi-Store E-commerce
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=your_stripe_key
NEXTAUTH_SECRET=your-nextauth-secret
NEXTAUTH_URL=http://localhost:3000
```

Start the development server:

```bash
npm run dev
```

Start the Django development server:

```bash
python manage.py runserver
```

Your application will be running at http://localhost:3000

### 4. Database Setup

Create the MySQL database:

```bash
CREATE DATABASE multistore_ecommerce;
```

The Spring Boot application will automatically create the necessary tables on
startup.
